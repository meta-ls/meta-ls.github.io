What is it special about MetaLS ?

Discover what MetaLS has to offer !
1.16.5

MetaLS is a free and open source server software using the GPL license. Making it open-source allows anyone to contribute to the project! You are free to create pull-requests on our github repository, or you can join our development team!

Easy To join

With a complete documentation, an easy installation and an active support, using Mohist is very easy and simple! Need help? Our community can help you on our Discord !

Polygon Blockchain

Mohist is compatible with almost every forge mod, and bukkit/spigot plugin. If you find a mod or plugin with issues, let us know it by joining our Discord server or opening a GitHub issue, and we can add support for it !

Proximity chat

Mohist is programmed to be clean, optimized using Paper optimizations, and well performing for Minecraft 1.12.2, 1.16.5 and soon 1.18. More versions may be supported in the future.
